# Docker Interview Questions - Intermediate (20 Questions)

## 1. What are Docker Layers?

Docker images are built using multiple read-only layers. Each Dockerfile instruction creates a new layer.

---

## 2. What is Docker Cache?

Docker cache reuses previously built layers to speed up image builds and reduce build time.

---

## 3. What is a Bind Mount?

A bind mount maps a directory from the host machine into a container.

---

## 4. Difference Between Volume and Bind Mount?

| Volume                     | Bind Mount                   |
| -------------------------- | ---------------------------- |
| Managed by Docker          | Managed by Host              |
| More portable              | Host-dependent               |
| Recommended for production | Commonly used in development |

---

## 5. What is Docker Compose?

Docker Compose is a tool used to define and run multi-container Docker applications using a YAML file.

---

## 6. What is a Docker Compose File?

A YAML file (`docker-compose.yml`) that defines services, networks, and volumes for an application.

---

## 7. What is a Docker Network?

A Docker network enables communication between containers and external systems.

---

## 8. What are the Types of Docker Networks?

* Bridge Network
* Host Network
* None Network
* Overlay Network
* Macvlan Network

---

## 9. What is a Bridge Network?

The default network driver that allows containers on the same host to communicate.

---

## 10. What is Host Networking?

Host networking removes network isolation and allows a container to use the host's network stack directly.

---

## 11. What is the Difference Between EXPOSE and -p?

### EXPOSE

Documents the port used by the application.

### -p

Maps container ports to host ports.

---

## 12. What is Docker Restart Policy?

A restart policy automatically restarts containers when they stop unexpectedly.

### Common Policies

* no
* always
* on-failure
* unless-stopped

---

## 13. What is Docker Inspect?

Docker Inspect provides detailed information about containers, images, volumes, and networks in JSON format.

---

## 14. What is Docker Logs?

Docker Logs displays the standard output and error logs generated by a container.

---

## 15. What is Docker Exec?

Docker Exec runs commands inside an already running container.

---

## 16. What is Docker Attach?

Docker Attach connects your terminal directly to a running container's main process.

---

## 17. Difference Between Docker Attach and Docker Exec?

| Attach                       | Exec                 |
| ---------------------------- | -------------------- |
| Connects to main process     | Starts a new process |
| Less flexible                | More commonly used   |
| Can affect container process | Safer option         |

---

## 18. What is Environment Variable Injection in Docker?

Environment variables allow configuration values to be passed into containers during runtime.

### Example

```bash
docker run -e APP_ENV=production myapp
```

---

## 19. What is Docker Registry Authentication?

Authentication allows users to securely push and pull images from private registries.

### Example

```bash
docker login
```

---

## 20. What is the Difference Between COPY and ADD?

| COPY                 | ADD                   |
| -------------------- | --------------------- |
| Copies files only    | Copies files and URLs |
| Preferred            | Extra functionality   |
| Predictable behavior | Less commonly used    |

### Interview Tip

Use `COPY` whenever possible and use `ADD` only when its additional features are required.
