# 🛡️ Fraud Detection System

## 🚀 Overview

Fraud Detection System is an Artificial Intelligence project that identifies suspicious financial transactions and calculates fraud risk scores.

The system analyzes transaction patterns, amount thresholds, and behavioral indicators to detect potentially fraudulent activities before financial losses occur.

---

## ✨ Features

* Transaction Monitoring
* Risk Score Generation
* Fraud Detection
* Suspicious Activity Analysis
* Financial Security Insights
* Interactive Visualization
