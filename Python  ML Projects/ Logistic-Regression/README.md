# 🎯 Logistic Regression From Scratch

A Machine Learning project that implements Logistic Regression using pure Python and NumPy without Scikit-Learn.

---

# 🚀 Project Overview

This project predicts whether a student will PASS or FAIL based on the number of study hours.

Unlike Linear Regression, Logistic Regression is used for Classification problems.

Example:

Study Hours → Pass / Fail

Email → Spam / Not Spam

Transaction → Fraud / Genuine

Customer → Churn / Not Churn

---

# 🎯 Objective

Build a Binary Classification Model from scratch using:

* Sigmoid Function
* Gradient Descent
* Binary Cross Entropy Loss
* Probability Estimation
