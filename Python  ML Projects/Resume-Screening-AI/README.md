# 📄 Resume Screening AI

## 🚀 Overview

Resume Screening AI is an AI-powered recruitment assistant that automatically analyzes resumes, matches candidate skills with job requirements, and generates a hiring recommendation.

The project demonstrates how Artificial Intelligence can streamline the recruitment process by reducing manual resume screening effort and improving candidate shortlisting accuracy.

---

## ✨ Features

* Resume Skill Extraction
* Job Description Analysis
* Skill Matching Engine
* ATS Score Generation
* Candidate Ranking
* Hiring Recommendation
