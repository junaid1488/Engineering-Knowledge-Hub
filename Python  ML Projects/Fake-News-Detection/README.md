# 📰 Fake News Detection

## 🚀 Overview

Fake News Detection is an Artificial Intelligence project that analyzes news content and predicts whether an article is likely to be genuine or misleading.

The system evaluates textual patterns and suspicious keywords to classify news articles into Real News or Fake News categories.

This project demonstrates how AI and Natural Language Processing can be used to combat misinformation and improve information reliability.
