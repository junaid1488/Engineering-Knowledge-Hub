# 🎤 AI Interview Analyzer

## 🚀 Overview

AI Interview Analyzer is an intelligent recruitment system that evaluates candidate interview responses and generates performance insights based on technical knowledge and communication skills.

The project simulates an AI-powered interview assessment platform used by modern hiring teams to improve recruitment efficiency and candidate evaluation.

---

## ✨ Features

* Interview Response Analysis
* Technical Skill Evaluation
* Communication Skill Assessment
* Candidate Scoring
* Hiring Recommendation
* Performance Visualization
