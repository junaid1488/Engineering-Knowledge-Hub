# 📈 Linear Regression From Scratch

A Machine Learning project that implements Linear Regression using pure Python and NumPy without relying on Scikit-Learn.

---

# 🚀 Project Overview

This project predicts student performance based on study hours using Gradient Descent optimization.

The model learns the relationship between:

Hours Studied → Exam Score

and generates future predictions automatically.

---

# 🎯 Objective

Build a Linear Regression model from scratch and understand:

* Machine Learning Fundamentals
