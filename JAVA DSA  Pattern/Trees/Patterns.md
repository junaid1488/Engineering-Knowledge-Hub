# 🌳 Trees Patterns

Trees are one of the most important interview topics.

---

# 1️⃣ DFS (Depth First Search)

Traversal Types:

- Preorder
- Inorder
- Postorder

Examples:

- Maximum Depth
- Path Sum
- Same Tree

Complexity:

Time: O(n)
Space: O(h)

---

# 2️⃣ BFS (Level Order)

Uses Queue.

Examples:

- Level Order Traversal
- Right Side View
- Zigzag Traversal

Complexity:

Time: O(n)
Space: O(n)

---

# 3️⃣ BST Properties

For BST:

Left < Root < Right

Examples:

- Validate BST
- Kth Smallest
- Closest Value

---

# 4️⃣ Tree Construction

Examples:

- Build Tree from Traversals
- Serialize & Deserialize

---

# 5️⃣ Lowest Common Ancestor

Examples:

- LCA in Binary Tree
- LCA in BST

---

# 6️⃣ Tree DP

Examples:

- Binary Tree Cameras
- Maximum Path Sum
- House Robber III
