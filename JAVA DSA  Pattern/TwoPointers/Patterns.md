# 👆👇 Two Pointers Pattern

Two Pointers is a technique that uses two indices to traverse a data structure efficiently.

Instead of nested loops, we move pointers strategically.

---

# 1️⃣ Opposite Direction

### Pattern

```text
left  → ←  right
```

### Complexity

```text
Time  : O(n)
Space : O(1)
```

### Examples

* Valid Palindrome
* Container With Most Water
* Trapping Rain Water

---

# 2️⃣ Same Direction

### Pattern

```text
slow →→→
fast →→→→→→
```

### Complexity

```text
Time  : O(n)
Space : O(1)
```

### Examples

* Move Zeroes
* Remove Duplicates
* Sort Colors

---

# 3️⃣ Sorted Array Two Pointers

### Pattern

```text
left + right
```

Move pointers based on comparison.

### Examples

* Two Sum II
* 3Sum
* 4Sum

---

# 4️⃣ Fast & Slow Pointer

### Pattern

```text
slow = 1 step

fast = 2 steps
```

### Examples

* Linked List Cycle
* Find Middle Node

---
