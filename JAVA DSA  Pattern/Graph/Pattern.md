# 📊 Graph Patterns

Graphs are among the most important interview topics.

---

# 1️⃣ DFS (Depth First Search)

Uses Recursion or Stack.

Applications:

* Connected Components
* Path Finding
* Cycle Detection

Examples:

* Number of Provinces
* Number of Islands

Complexity:

Time: O(V + E)

---

# 2️⃣ BFS (Breadth First Search)

Uses Queue.

Applications:

* Shortest Path (Unweighted)
* Multi-source BFS

Examples:

* Rotting Oranges
* Word Ladder

Complexity:

Time: O(V + E)

---

# 3️⃣ Topological Sort

Used for Directed Acyclic Graphs.

Applications:

* Course Scheduling
* Dependency Resolution

Examples:

* Course Schedule
* Alien Dictionary

Complexity:

Time: O(V + E)

---

# 4️⃣ Union Find (DSU)

Applications:

* Connected Components
* Cycle Detection

Examples:

* Number of Provinces
* Redundant Connection

Complexity:

Nearly O(1)

---

# 5️⃣ Dijkstra

Applications:

* Shortest Path in Weighted Graph

Examples:

* Network Delay Time
* Swim In Rising Water

Complexity:

O(E log V)

---

# 6️⃣ Tarjan Algorithm

Applications:

* Bridges
* Articulation Points

Examples:

* Critical Connections

Complexity:

O(V + E)

---
