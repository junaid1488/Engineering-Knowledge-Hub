# 🤖 ML Algorithms From Scratch

> A collection of Machine Learning algorithms implemented entirely using **NumPy**, without relying on Scikit-Learn.

This repository focuses on understanding:

* 📖 Theory
* 🧮 Mathematical Foundations
* ⚡ NumPy Implementation
* 🧪 Example Usage
* 📈 Time Complexity
* 🎯 Real-world Applications

The goal is to learn Machine Learning from first principles.

# 🎯 Why Build From Scratch?

Most developers can use:

```python
from sklearn.linear_model import LinearRegression
```

Few developers can explain:

* How Gradient Descent works
* How Entropy is calculated
* How PCA finds principal components
* How SVM maximizes margins
* How Naive Bayes computes probabilities

This repository focuses on understanding the internals.
